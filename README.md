\*\*DropShelf — 菜单栏临时文件暂存（macOS）

一个轻量级的 macOS 菜单栏“暂存”工具，支持将文件拖入到菜单栏图标或弹出面板中暂存，并能从面板拖出文件到其他应用（正在完善）。界面使用系统的原生毛玻璃（vibrancy）效果以获得与系统菜单一致的外观。

**特性**:

- **菜单栏图标**: 支持拖拽文件到菜单栏图标直接暂存。
- **原生毛玻璃弹出面板**: 使用 `NSVisualEffectView` 承载 SwiftUI 内容以匹配系统菜单的视觉效果。
- **多文件暂存**: 列表展示已暂存文件，可逐一删除。
- **拖入/拖出**: 支持从 Finder 或其他应用拖入文件；拖出功能已实现基础逻辑（文件名保留通过文件承诺/Promise，可进一步兼容更多目标应用）。
- **交互反馈**: 成功添加时提供短暂视觉与触觉反馈（macOS 支持时）。
- **持久化弹出**: 弹出面板为应用定义行为（再次点击状态栏图标才关闭）。

**Quick Start（Xcode）**:

1. 打开 Xcode，选择 `Open...`，打开工作区 `DropShelf/DropShelf.xcodeproj`。
2. 选择目标为 `DropShelf`，构建并运行（macOS 机型）。
